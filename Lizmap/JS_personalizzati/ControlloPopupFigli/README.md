# CONTROLLO DEI POPUP DEI FIGLI
Questo Tool permette di controllare la visibilità   
dei popup dei figli in una relazione impostata in QGIS,   
anche quando i popup sono formattati con HTML

## V01 - VERSIONE PER LIZ 3.8 
Versione usata e stabile fino al 18.01.2026
Il Tool è composto da 4 file:   
1) creare il bottone nel popup padre (html)
2) ceare il bottone nel popup figli (html)
3) incollare le regole css in coda al file map.css
4) salvare il javascript

Da testare con Lizmap 3.9.
