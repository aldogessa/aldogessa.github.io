# GENERAZIONE CDU
Questo script, a partire da una selezione di record dalla tabella padre, apre la tabella figlia con   
i record filtrati sulla base di una relazione definita con chiave esterna, senza la necessità di   
definire una relazione esplicità nelle proprietà del progetto QGIS.   
La tabella padre e figlia possono anche coincidere, come nel caso dello script esposto.

## Versione QGIS testata
Lo script è testato nella versione QGIS 3.28 per le versioni successive potrebbero essere
necessari alcuni aggiustamenti.
